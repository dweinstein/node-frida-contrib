# SYNOPSIS

A collection of utilities that use [frida](http://www.frida.re/)

# INSTALL

```
npm install -g frida-contrib
```

# USAGE

## common

### LIST APPS

```
± frida-list-apps
```

## iOS

### DUMP UI ELEMENTS
Dump the serialized UI.

```shell
± frida-ios-dump-ui | gist
https://gist.github.com/7ec87584eebfd81e734f
```
